# Райский сад Сибири — WordPress сайт-каталог

Верстка и интеграция сайта-каталога растений в WordPress.

## О проекте

Сайт-каталог "Райский сад Сибири".  
Проект включает адаптивную верстку страниц и интеграцию верстки в WordPress-тему.

## Что реализовано

- адаптивная верстка страниц
- интеграция верстки в WordPress
- разработка шаблонов темы
- вывод карточек каталога
- работа с пользовательскими полями
- организация структуры каталога

## Технологии

- HTML5
- SCSS
- JavaScript
- WordPress
- ACF
- BEM

## Роль в проекте

- разработка верстки
- интеграция верстки в WordPress
- настройка структуры каталога
