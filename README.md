# Райский сад Сибири — WordPress сайт-каталог

Верстка и интеграция сайта-каталога растений в WordPress.

Сайт: райскийсадсибири.рф или https://xn--80aadlscbacbj3ekfdc.xn--p1ai/ 

## О проекте

Сайт-каталог "Райский сад Сибири".  
Проект включает адаптивную верстку страниц и интеграцию верстки в WordPress-тему.

## Функциональность

- Каталог товаров с карточками и детальными страницами
- Страницы категорий (таксономии)
- Хлебные крошки для навигации
- Динамический вывод данных из админ-панели
- Адаптивная верстка всех страниц

## Работа с WordPress

- Разработка кастомной темы
- Создание кастомных типов записей
- Настройка и вывод таксономий
- Работа с WP Query
- Интеграция и вывод данных через ACF
- организация структуры каталога

## Что реализовано: 

- Гибкая система шаблонов (single, archive, taxonomy)
- Условный вывод контента (скрытие блоков при отсутствии данных)
- Оптимизация изображений (адаптивные размеры, корректный вывод)
- Корректная работа сайта на разных устройствах

## Роль в проекте

- разработка верстки
- интеграция верстки в WordPress
- настройка структуры каталога
